# transformers_expand

基于`transformers`拓展的常用任务训练集合与模型集合

* 整体结构继承自`transformers`
* **基于`Pytorch`构建，目前定义的所有模型及Trainer只能基于`Pytorch`运行**

## 安装
* 未上传至PyPi, 需要从git安装
```bash
pip install git+https://github.com/casuallyName/transformers_expand.git
```
